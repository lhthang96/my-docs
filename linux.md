# LINUX TIPS

## Vim to edit files

```
(sudo) vim <file_name>
```

- `insert`: start editing file
- `esc`: escape editing
- `:q`: quit vim
- `:x`: save and quit vim
- `:x!`: an exclamation to override read-only mode.

## Other tips

- `htop` to monitor system processes and see how much resources is being used.
